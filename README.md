\# Real-Time Emotion Detection



A real-time facial emotion detection system built with CNN trained on FER-2013 dataset.



\## Tech Stack

\- Python

\- TensorFlow / Keras

\- OpenCV

\- Streamlit

\- FER-2013 Dataset



\## Emotions Detected

Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral



\## Project Structure

\- train\_model.py — CNN architecture and training

\- test\_webcam.py — Real-time webcam inference

\- app.py — Streamlit web application



\## How to Run

1\. Install dependencies: pip install -r requirements.txt

2\. Train model: python train\_model.py

3\. Run app: streamlit run app.py
